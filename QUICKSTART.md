# 5 分钟上手

1. **安装 Ollama** 并执行：`ollama pull llama3`、`ollama pull nomic-embed-text`  
2. **安装依赖**：`pip install -r requirements.txt`  
3. **配置**：复制 `.env.example` 为 `.env`（一般无需改）  
4. **启动**：在项目根目录执行 `start.bat`（Windows）或 `./start.sh`（Unix）  
5. 浏览器打开 `http://localhost:8501`，创建角色后开始探索；点「与××交谈」进入对话  

若未启动 Ollama，游戏仍可使用离线降级文本运行。
