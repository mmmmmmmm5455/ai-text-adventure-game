# 奪舍系统使用指南（离线模式）

**适用场景：** 当无法连接 PostgreSQL 数据库时，使用离线模式体验奪舍功能

---

## 🎭 什么是奪舍系统？

奪舍（Possession）是游戏的核心创新机制之一：
- 玩家可以保存角色快照
- 其他玩家可以奪舍这个角色
- 接管角色的身份、进度、物品、任务
- 继续完成原角色的故事

---

## 🚀 快速开始

### 方法 1：使用命令行工具（推荐）

```bash
# 1. 进入游戏目录
cd ai_text_advanture_game

# 2. 查看所有可奪舍的角色
python tools/possession_cli.py list

# 3. 查看角色详情
python tools/possession_cli.py show predefined_001

# 4. 奪舍角色
python tools/possession_cli.py claim predefined_001 --name "你的名字" --gender "性别"

# 5. 启动游戏
python -m streamlit run frontend/app.py
# 或双击 start.bat
```

### 方法 2：在游戏内奪舍

```
1. 启动游戏
2. 进入"存档"界面
3. 点击"奪舍系统"
4. 从预设列表中选择角色
5. 输入你的名字
6. 开始游戏
```

## 💡 使用技巧

### 选择适合的角色

**如果你是新手：**
- 推荐：1（铁血战士）或 4（银月吟游诗人）
- 原因：平衡性好，不会太难

**如果你喜欢战斗：**
- 推荐：1（铁血战士）或 5（废土医生）
- 原因：高血量，容错率高

**如果你喜欢魔法：**
- 推荐：2（奥术师）
- 原因：高智力，魔法力强

**如果你喜欢潜行：**
- 推荐：3（暗影盗贼）
- 原因：高敏捷，高价值物品

**如果你想体验完整游戏：**
- 推荐：5（废土医生）
- 原因：3章，内容丰富

### 不同章节的特点

**第1章（早期）：**
- ✅ 简单易懂
- ✅ 适合新手
- ✅ 探索村庄周边

**第2章（中期）：**
- ⚠️ 难度提升
- ✅ 适合有经验的玩家
- ✅ 内容更丰富

**第3章（后期）：**
- ⚠️ 难度较高
- ✅ 需要策略
- ✅ 高价值奖励

## 🎮 奪舍后的体验

### 你会继承：

✅ **角色属性**
- 名字、职业、等级
- HP、MP、金币
- 6个属性值

✅ **角色背景**
- 背景故事
- 特质（正面/负面）
- 最后的遗言

✅ **游戏进度**
- 当前场景
- 已解锁的地点
- 进行中的任务
- 物品和装备

### 体验内容：

1. **继续原任务**
   - 完成原角色未完成的任务
   - 探索未发现的场景
   - 发现未完成的剧情

2. **新故事线**
   - 以新身份重新体验
   - 不同的选择可能
   - 独特的角色发展

3. **多样化玩法**
   - 不同职业的战斗风格
   - 不同背景的互动方式
   - 不同特质的特殊效果

### claim - 奪舍快照

```bash
python tools/possession_cli.py claim predefined_001 --name "你的名字" --gender "性别"
```

**参数：**
- `snapshot_id`：快照ID（如 predefined_001）
- `--name`：你的新名字（可选，默认"新玩家"）
- `--gender`：性别（可选，默认"男"）

**示例：**
```bash
# 奪舍为"铁血战士"
python tools/possession_cli.py claim predefined_001 --name "我的角色"

# 奪舍为"奥术师"，女性
python tools/possession_cli.py claim predefined_002 --name "艾莉西亚" --gender "女"
```

---

python tools/possession_cli.py my offline_player_predefined_001

## 🐛 故障排除

### 问题：找不到快照

**原因：**
- 输入了错误的快照ID
- 使用了数字而不是完整ID

**解决：**
```bash
# 使用完整ID
python tools/possession_cli.py show predefined_001

# 或者先列出所有快照
python tools/possession_cli.py list
```

---

### 问题：快照已被奪舍

**原因：**
- 其他玩家已经奪舍了这个快照
- 在数据库模式下，每个快照只能被奪舍一次

**解决：**
```bash
# 查看其他可奪舍的快照
python tools/possession_cli.py list

# 选择其他快照奪舍
python tools/possession_cli.py claim predefined_002
```

---

### 问题：角色创建失败

**原因：**
- 游戏版本与快照不兼容
- 缺少必要的依赖

**解决：**
1. 确保使用最新的游戏版本
2. 检查所有依赖是否安装
3. 查看 README.md 了解环境要求
