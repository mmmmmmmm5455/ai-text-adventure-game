# 国际化（i18n）预留策略

## 当前状态

- 界面与游戏内提示以**简体中文**为主；引擎与剧情模块注释可为中文。

## 若后续支持多语言

1. **UI 字符串**：集中为字典或 `gettext`/`babel`，按 `locale` 切换；Streamlit 侧用 session 或 query 参数选择语言。
2. **生成内容**：Prompt 模板按语言分文件（如 `prompts/zh/`, `prompts/en/`），由 `PromptPolicy` 根据设置加载。
3. **剧情资产**：外置 JSON/YAML 时，字段级 `name.zh` / `name.en` 或分文件 `scenes_zh.json`，版本号仍走 `STORY_ASSET_VERSION` 与 `docs/STORY_VERSIONING.md`。

## 与存档

- 存档内玩家输入与日志可能混合语言；`schema_version` 不负责语言，仅负责结构迁移。
