# API 说明（模块级）

- `engine.story_engine.StoryEngine`：场景生成、选项生成、`process_choice`  
- `engine.ai_dialogue.AIDialogueEngine`：`start_dialogue` / `continue_dialogue` / `end_dialogue`  
- `engine.memory_manager.MemoryManager`：`add_memory` / `query_relevant`  
- `game.game_state.GameState`：`save` / `load`、`to_dict` / `from_dict`  

详细请直接阅读源码中的类型注解与中文注释。
