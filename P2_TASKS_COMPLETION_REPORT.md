## ✅ P2-11: 存档标签和描述系统

### 问题
存档只能保存，无法添加标签和描述，不便于管理多个存档。

### 解决方案

#### 1. 创建存档元数据系统
**文件：** `game/save_metadata.py`

**核心功能：**
- `SaveMetadata` 类：存储标签、描述、创建/更新时间
- `load_metadata()`：加载存档元数据
- `save_metadata()`：保存存档元数据
- `get_suggested_tags()`：根据存档内容推荐标签

#### 2. 15 个预定义标签
```
1. 主线任务
2. 支线任务
3. 探索
4. 战斗
5. 剧情分支
6. 低等级
7. 中等级
8. 高等级
9. 装备齐全
10. 资金充足
11. 特殊事件
12. 存档点
13. 重要选择
14. 速通
15. 完美通关
```

#### 3. 智能标签推荐
根据存档内容自动推荐标签：
- **任务类型：** 主线任务、支线任务
- **等级：** 低等级（≤5）、中等级（6-10）、高等级（>10）
- **资金：** 资金充足（金币 > 100）
- **装备：** 装备齐全（已装备武器或护甲）

#### 4. UI 改进
**文件：** `frontend/screen/archive.py`

**界面布局：**
- **左栏：** 存档操作（保存/加载）
- **右栏：** 元数据管理
  - 显示当前标签和描述
  - 编辑元数据（展开式）
  - 自定义标签输入
  - 推荐标签显示
